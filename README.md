# iTUG Dashboard Next.js

Proyecto Next.js basico para visualizar el dashboard iTUG comparando a Sofía con un grupo de control.

## Ejecutar

```bash
npm install
npm run dev
```

Luego abre `http://localhost:3000`.

## Estructura

- `app/`: App Router de Next.js, layout, pagina inicial y estilos globales.
- `components/dashboard/`: secciones visuales del dashboard.
- `data/`: senales de movimiento, ensayos y sujetos.
- `lib/`: colores y helpers estadisticos.

El archivo original `itug_dashboard.jsx` se conserva como referencia.
